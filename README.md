# 🚌 MERN Bus Seat Booking System

A full-stack bus seat booking web application built using the MERN stack.

## 🚀 Features

- User & Admin Authentication (JWT)
- Role-Based Access Control
- Multi-Seat Selection with Toggle
- Seat Locking System
- Booking & Cancellation
- Admin Dashboard with Statistics
- Modern Responsive UI
- MongoDB Database Integration

## 🛠 Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js
- JWT Authentication

## ⚙️ Setup Instructions

### Backend

cd backend  
npm install  
npm run dev  

### Frontend

cd frontend  
npm install  
npm start
